# Model artifacts directory
# ==========================
# Direktori ini diisi secara otomatis oleh train.py
#
# File yang akan dihasilkan setelah training:
#   - best_model.pkl          : Model terbaik (MNB atau SVM)
#   - mnb_model.pkl           : Multinomial Naive Bayes
#   - svm_model.pkl           : SVM Linear Kernel
#   - tfidf_vectorizer.pkl    : TF-IDF Vectorizer
#   - training_metadata.json  : Info & metrik training
#   - reports/                : Grafik & laporan evaluasi
#
# Jalankan: python train.py
